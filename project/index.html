Hugh Roper: Classical Harpist

Site Name

The name “Classical Harpist” clearly identifies the website topic and supports search terms related to harp lessons and performances.

Purpose

The purpose of this website is to introduce visitors to Michele Roper, a classically trained harpist and instructor in New Mexico. The site will provide information about Michele’s musical background, upcoming performances, and teaching opportunities. Visitors will be able to view a schedule of events, learn more about the musician, and submit a form to join a student wait list. The website is intended to serve current and prospective students as well as classical music enthusiasts interested in local performances.

Target Audience

  • Current harp students and their families
  • Prospective students interested in harp lessons
  • Parents researching music instruction opportunities
  • Classical music enthusiasts interested in local performances
  • Local educators, musicians, and arts organizations

Typography

Heading Font: Source Sans 3 (Weight 200)

Used for page headings and section titles. The light font weight creates an elegant and refined appearance that complements the classical harp theme.

Body Font: Source Sans 3 (Weight 400)

Used for paragraph text and general content. The regular weight provides clear readability while maintaining a clean and professional look.

Content Plan

Home Page

Introduction to Michele Roper, a classically trained harpist and instructor. This page will include a brief biography, highlights of upcoming events, current weather conditions, and links to other sections of the website.

About Page

Information about Michele's musical background, classical training, teaching philosophy, and involvement in the local arts community.

Events Page

A list of upcoming performances and events loaded from a local JSON file. Event information will include dates, locations, and descriptions.

Student Wait List Page

A form allowing prospective students to request information and join a wait list for future lesson openings. A thank-you page will confirm successful submission.

SEO

Meta Description

Explore classical harp lessons, performances, and upcoming events with Michele Roper, a classically trained harpist serving Los Alamos, New Mexico.

Keywords

classical harpist, harp lessons, harp teacher, Michele Roper, Los Alamos, New Mexico, classical harp, music lessons, harp performance, harp instructor, classical music

Dynamic Elements/APIs

The Events page will use JavaScript to load upcoming performances from a local JSON file. Each event will include information such as the event name, date, location, and description.

The Home page will use a weather API to display current weather information for Los Alamos, New Mexico.

The Student Wait List page will include a form where prospective students can submit their name, contact information, and lesson interest. After submitting the form, users will be directed to a thank-you page.
